## Getting Started

To get a local copy up and running follow these simple example steps.

### Prerequisites

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/tatsmaki/chess.git
   ```
2. Install NPM packages
   ```sh
   yarn install
   ```
3. Run application
   ```sh-
   yarn start
   ```
